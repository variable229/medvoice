Signing Up and Logging In

MedVoice strives to raise health awareness with social support. To be a part of the MedVoice community, simply launch the mobile application and select the 'Sign Up' button at the bottom right of the application. After signing up, you will automatically be taken to the MedVoice dashboard where you will notice several features such as Journal, P2P, Chart, Scan, and News. Learn more about these features in the navigation bar above. To learn more about the Dashboard, check out the links above.
